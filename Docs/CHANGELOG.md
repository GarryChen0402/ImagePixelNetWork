### 核心设计决策

- **网络结构**：编码器-解码器 + 可微分调色板量化层
- **量化方式**：温度退火软分配，训练中从 τ=1.0 退火至 τ=0.1
- **损失函数**：内容损失 + 风格损失 + TV 损失 + 调色板均衡损失
- **数据策略**：合成数据为主（10K+），真实像素艺术数据为辅（1K+）
- **训练策略**：三阶段渐进式训练（预训练 → 风格注入 → 精细调优）
